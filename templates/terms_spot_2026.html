業務委託基本契約約款
（スポット契約用・2026年改正法対応版）
本約款は、発注者が発行する発注書（以下「本発注書」という。）に基づき、発注者が受注者に対し業務を委託し、受注者がこれを受託する取引（以下「本契約」という。）に適用される条件を定めるものである。
第1条（総則および契約の成立）
  1. 発注者および受注者は、対等な立場で、互いに協力し、信義を守り、誠実に本契約を履行するものとする。
  2. 本契約は、発注者が本発注書を発行（電子メールやクラウドサービス等による交付を含む。）し、受注者が以下のいずれかの方法により承諾の意思表示をした時点で成立するものとする。
    • (1) 本発注書受領後5営業日以内に、書面または電子メール等による承諾通知を行うこと
    • (2) 本発注書に記載された業務に着手すること
    • (3) その他、受注者による明確な承諾の意思表示があること
  3. 前項にかかわらず、受注者が本発注書受領後5営業日以内に異議を申し出ない場合は、承諾したものとみなす。なお、発注者は、受注者の承諾（みなし承諾を含む。）前に限り、本発注を撤回することができる。
  4. 本発注書および本約款の交付については、法令の定める範囲において、電子メール等の電磁的方法により行うことができる。
  5. 発注者は、法令に基づき、本発注書において、業務内容、報酬額、支払期日その他取引条件を明確に記載し、受注者に対して書面または電磁的方法により交付するものとする。
第2条（業務の遂行および独立性）
  1. 受注者は、本発注書および仕様書（以下「仕様書等」という。）に従い、善良なる管理者の注意をもって誠実に業務を遂行するものとする。
  2. 受注者は、独立した事業者として自己の裁量と責任に基づき業務を遂行し、発注者の指揮命令下には置かれないものとする。本契約は、雇用契約またはこれに類する契約を含まないことを相互に確認する。
  3. 受注者は、業務の遂行にあたり、自己の責任において必要な人員、設備、環境を用意するものとする。
第3条（報酬の決定および協議）
報酬額は、仕様書等に定める通りとする。ただし、発注者は報酬額の決定にあたり、受注者からの協議の求めがあった場合には誠実に応じるものとし、一方的に不当に低い対価を定めてはならない。
第4条（成果物の納品および検収）
  1. 業務に成果物が伴う場合、受注者は仕様書等に定める納期および方法に従い、当該成果物を発注者に納入（以下「納品」という。）するものとする。
  2. 発注者は、成果物の納品後、仕様書等に定める期間内（特段の定めがない場合は10営業日以内）に検査を行い、その合否（以下「検収」という。）を受注者に通知するものとする。
  3. 前項の期間内に発注者から何らの通知もなされない場合、当該成果物は検収に合格したものとみなす。
  4. 成果物の所有権および危険負担は、検収合格時（または検収合格とみなされた時点）をもって受注者から発注者に移転する。
第5条（契約不適合責任）
  1. 納品された成果物または遂行された業務の内容が、種類、品質または数量に関して本契約の内容と適合しない場合（以下「契約不適合」という。）、発注者は受注者に対し、相当の期間を定めて履行の追完（修補、代替物の引渡し等）を請求することができる。
  2. 前項の規定は、成果物の検収完了後または業務終了後6か月以内に発注者が契約不適合を知り、その旨を受注者に通知した場合に限り適用されるものとする。ただし、受注者が悪意または重過失により契約不適合を知っていた場合はこの限りではない。
  3. 準委任契約（成果物の完成を目的としない業務）の場合、受注者の業務遂行に善管注意義務違反等の重大な不備があったときは、発注者は報酬の減額または損害賠償を請求することができる。
  4. 発注者は、受注者に責任がないにもかかわらず、発注後に不当に報酬を減額し、または受領を拒否・返品してはならない。
第6条（報酬および支払条件）
  1. 発注者は、受注者に対し、仕様書等に定める報酬額（消費税別）を支払うものとする。
  2. 報酬の支払期日は、以下の区分に従うものとする。
    • (1) 受注者が「特定受託事業者（フリーランス法適用）」または「中小受託事業者（取適法適用）」に該当する場合
      成果物の受領日（役務提供の場合は業務終了日）から起算して60日以内、かつ発注書に記載された支払期日までのいずれか早い日とする。
    • (2) 上記以外の場合
      発注書に記載された支払期日（ただし、検収完了日から90日以内）とする。
  3. 報酬の支払は、原則として銀行振込により行うものとし、手形その他支払期日までに代金相当額を得ることが困難な手段は用いない。振込手数料は発注者の負担とする。
  4. 発注者は、報酬を支払う際、法令に基づき源泉徴収税額を控除することができる。
  5. 発注者が報酬の支払を遅延した場合、受注者は以下の年率で遅延損害金を請求することができる。
    • (1) フリーランス法または取適法適用対象の場合：年14.6％
    • (2) 上記以外の場合：年3.0％
第7条（知的財産権の帰属）
成果物に関する著作権（著作権法第27条および第28条の権利を含む。）およびその他知的財産権の帰属については、仕様書等の記載に従い、以下のいずれかが適用される。
  1. 【譲渡型】（特段の定めがない場合の原則）
    成果物の納品完了および報酬の完済をもって、知的財産権は受注者から発注者に移転する。受注者は著作者人格権を行使しないものとする。ただし、成果物に第三者の権利が含まれる場合や、受注者が従前から保有する権利が含まれる場合は、事前に発注者に通知し、その取扱いを協議するものとする。
  2. 【利用許諾型】
    知的財産権は受注者に留保される。受注者は発注者に対し、仕様書等に定める範囲（利用目的、期間、地域等）で当該成果物の利用を許諾する。
第8条（成果物の知的財産権に関する表明保証）
  1. 受注者は、成果物（仕様書等に基づき受注者が作成・提供する一切の成果物、資料、データ、プログラム、デザイン等を含む。以下同じ。）について、発注者が本契約に基づき取得または利用する権利を妨げる第三者の権利が存在しないこと、または存在する場合には事前に発注者へ書面により通知し、必要な権利処理を行うことを表明し、保証する。
  2. 受注者は、成果物が第三者の著作権、著作者人格権、商標権、意匠権、特許権、営業秘密その他一切の権利を侵害しないことを表明し、保証する。
  3. 受注者は、成果物に第三者が権利を有する素材（画像、フォント、音源、ライブラリ、テンプレート、生成AI出力物を含むがこれらに限られない。）を含める場合、発注者による利用態様に照らして必要かつ十分な利用許諾・権利処理を事前に行い、その内容を発注者に説明できる状態にするものとする。
  4. 前各項に違反し、発注者または発注者の関係者に対し第三者から請求、申立て、差止め、その他の紛争（以下「紛争等」という。）がなされた場合、受注者は自己の費用と責任でこれを解決し、発注者に生じた損害（合理的な弁護士費用を含む。）を賠償する。ただし、当該紛争等が第5項に定義する発注者指定素材に起因する場合はこの限りでない
  5. 発注者が、受注者に対し本業務の遂行のため提供し、又は利用を指定する素材（ロゴ、商品画像、原稿、既存動画、音源その他一切を含む。以下「発注者指定素材」という。）に関する著作権その他一切の権利は、発注者又は発注者が正当な権利を有する第三者に帰属する（発注者指定素材は発注者著作）。 受注者は、本業務の遂行に必要な範囲で、発注者指定素材を無償で使用できる。
  6. 発注者は、発注者指定素材について、受注者による前項の使用および発注者による本契約上の利用（発注者の利用態様を含む。）に必要な権利を有し、又は正当な許諾を得ていることを表明し、保証する。 発注者指定素材に起因して紛争等が生じた場合、発注者は自己の費用と責任でこれを解決し、受注者に生じた損害（合理的な弁護士費用を含む。）を賠償する。 ただし、受注者が発注者の指示を逸脱して発注者指定素材を改変し、又は本業務の範囲を超えて使用したことに起因する場合はこの限りでない。
第9条（秘密保持）
  1. 発注者および受注者は、本契約の履行過程で知り得た相手方の技術上・営業上その他の秘密情報を、相手方の書面による事前の承諾なく第三者に開示または漏洩してはならず、本契約の目的以外に使用してはならない。
  2. 前項の義務は、以下の情報には適用されない。
    • (1) 開示時点で既に公知であった情報
    • (2) 開示後、自己の責によらず公知となった情報
    • (3) 正当な権限を有する第三者から秘密保持義務を負わずに取得した情報
  3. 本条の義務は、本契約終了後も5年間（ただし個人情報については第10条に従う。）有効とする。
  4. 受注者が成果物を自身のポートフォリオやSNS等で実績として公開する場合は、必ず事前に発注者の承諾を得るものとし、公開可能時期は発注者が指定する日以降とする。
第10条（個人情報の取扱い／データプロテクション）
  1. 発注者および受注者は、本業務遂行にあたり個人情報（個人情報保護法その他適用法令に定める「個人情報」「個人データ」等を含む。）を取り扱う場合、個人情報保護法および関連法令を遵守し、適切に管理するものとする。
  2. 受注者は、個人情報を本業務の遂行に必要な範囲でのみ取り扱い、発注者の事前の書面による承諾なく、第三者へ提供、開示または漏えいしてはならない。
  3. 受注者は、個人情報の漏えい、滅失または毀損の防止その他個人情報の安全管理のために必要かつ適切な措置を講じ、従業者（再委託先を含む。）に対する必要かつ適切な監督を行うものとする。
  4. 受注者は、個人情報の取扱いに関し事故（漏えい等）が発生し、またはそのおそれがある場合、直ちに発注者へ報告し、発注者の指示に従い、必要な是正措置および再発防止措置を講じるものとする。
  5. 本契約終了後、受注者は発注者の指示に従い、個人情報を速やかに返却または安全に消去・廃棄し、その実施状況を合理的に証明できるものとする。
第11条（契約上の地位等の譲渡禁止）
発注者および受注者は、相手方の書面による事前の承諾なく、本契約に基づく契約上の地位または権利義務を第三者に譲渡、承継し、または担保に供してはならない。
第12条（未承諾の再委託禁止）
  1. 受注者は、本業務の全部または重要な一部を第三者に再委託する場合には、事前に発注者の書面による承諾を得るものとする。
  2. 受注者は、発注者の承諾を得て再委託を行う場合であっても、再委託先に対し本契約と同等以上の秘密保持義務および個人情報保護義務等を課し、必要かつ適切に監督するものとし、再委託先の行為について一切の責任を負う。
第13条（法令遵守およびハラスメント防止）
  1. 発注者は、受注者が法の定める対象事業者に該当する場合、「特定受託事業者に係る取引の適正化等に関する法律（フリーランス法）」および「製造委託等に係る中小受託事業者に対する代金の支払の遅延等の防止に関する法律（取適法）」を遵守し、適正な取引条件の設定、ハラスメント防止措置等を講じるものとする。
  2. 発注者は、受注者が安心して業務を遂行できるよう、業務に関連するハラスメントの防止および相談対応体制の整備に努めるものとする。

第14条（反社会的勢力の排除）
  1. 発注者および受注者は、現在および将来にわたり、自ら（役員、実質的支配者、主要な株主、従業員等を含む。）が反社会的勢力（暴力団、暴力団員、暴力団関係企業、総会屋、社会運動等標ぼうゴロ、特殊知能暴力集団その他これらに準ずる者をいう。以下同じ。）に該当しないこと、ならびに反社会的勢力と一切の関係（資金提供、便宜供与、取引関係、その他実質的な関与を含む。）を有しないことを表明し、保証する。
  2. 発注者および受注者は、反社会的勢力を利用しないこと、反社会的勢力に自己の名義を利用させないこと、ならびに自らまたは第三者を通じて、相手方に対して暴力的な要求行為、法的責任を超えた不当な要求行為、脅迫的言動または暴力を用いる行為、風説の流布・偽計または威力を用いて相手方の信用を毀損しまたは業務を妨害する行為等を行わないことを確約する。
  3. 相手方が前二項に違反した場合、当事者は何らの催告を要せず、直ちに本契約を解除することができる。
  4. 前項に基づき本契約が解除された場合、解除した当事者は、解除により相手方に生じた損害を賠償する責任を負わない。なお、相手方に損害が生じたときは、解除された当事者はその損害（合理的な弁護士費用を含む。）を賠償する。
第15条（解除および損害賠償）
  1. 発注者または受注者は、相手方が本契約に違反し、相当期間を定めた催告にもかかわらず是正されない場合、本契約を解除することができる。
  2. 損害賠償責任の上限等は、別途の定めがない限り、本契約に基づき支払われる報酬額（税抜）を上限とする。ただし故意または重大な過失による場合等は除く。
第16条（中途解約時の精算）
本契約が中途で解約された場合、発注者および受注者は、業務の進捗状況に基づき報酬額を協議の上精算するものとする。
第17条（準拠法および合意管轄）
本契約の準拠法は日本法とし、本契約に関する紛争については、東京地方裁判所を第一審の専属的合意管轄裁判所とする。
第18条（協議事項）
本契約に定めのない事項または疑義が生じた事項については、当事者が誠意をもって協議し、解決を図るものとする。
